## Installation

In the project directory, you can run:

###  `npm install`

To use the npm package

### `npm run dev`

Runs the proyect on browser from "frontend" folder.

### `npm run test:unit`

To run test from frontend folder.

Runs the proyect backend server 

### `mvn spring-boot:run`
